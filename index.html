Ejercicio clase 20 DWFSH1-108 - Pseudoclases CSS

  • Definición: Una pseudoclase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado.
  • Seleccionamos 10 pseudoclases aparte de las vistas en clase como: ":link, :visited, :hover, :active, :nth_child()."
  1. :checked
  2. :default
  3. :dir()
  4. :autofill
  5. :empty
  6. :enabled
  7. :root
  8. :any-link
  9. :placeholder-shown
  1. :checked

    2. Representa cualquier elemento de selección dentro de un formulario o página web.



  2. :default

    3. Representa cualquier elemento de formulario que sea el predeterminado entre un grupo de elementos relacionados.



  3. :dir()

    4. La pseudoclase :dir de CSS coincide con los elementos en función de la direccionalidad del texto contenido en ellos.

    Usa solo el valor semántico de la direccionalidad, es decir, el definido en el documento mismo. No tiene en cuenta la direccionalidad del estilo, es decir, la direccionalidad establecida por las propiedades de CSS como direction.

  4. :autofill

    5. La pseudoclase CSS coincide cuando un elemento input tiene su valor autocompletado por el navegador. La clase deja de coincidir si el usuario edita el campo.


  5. :empty

    6. Representa cualquier elemento que no tenga hijos. Los hijos pueden ser nodos de elemento o texto (incluido el espacio en blanco). Los comentarios o las instrucciones de procesamiento no afectan si un elemento se considera vacío.

    Voy a ser de color rosa.

  6. :enabled

    7. Representa cualquier elemento habilitado. Un elemento está habilitado si puede ser activado (es decir seleccionado, se puede hacer click en él, acepta que se le introduzca texto, etc.) o si accepta el foco. El elemento también tiene un estado deshabilitado el en cuál, no puede ser activado ni acepta el foco.




  7. :root

    8. La pseudoclase selecciona el elemento raíz de un árbol que representa el documento. En HTML, :root representa el elemento html y es idéntico al selector html, excepto que su especificidad es mayor.

    :root puede ser util para declarar variables CSS globales.

  8. :any-link

    9. La pseudo-clase :any-link de CSS representa a un elemento que actúa como el ancla origen de un hipervínculo independientemente de si ha sido visitado, es decir, coincide con cualquier elemento a, area o link con un atributo href. Por lo tanto, coincide con todos los elementos que coincidan con :link o :visited.

  9. :placeholder-shown

    10. La pseudo-clase :placeholder-shown de CSS representa cualquier elemento input o textarea que esté mostrando actualmente el texto de marcador de posición placeholder.
